## Overview

PORMAZION empowers you to create, organize, and collaborate on project documentation with ease. You get a centralized space to manage docs for multiple projects, ensuring your team stays aligned without scattered files or version chaos. Build knowledge bases, API references, and user guides that scale with your needs.

Key benefits include real-time collaboration, version history, and search across all your spaces. PORMAZION uses brand color `#3B82F6` for consistent theming—apply it in custom CSS or logos for a professional look.

<Callout kind="info">
  PORMAZION suits engineering teams, product managers, and support staff who need structured, searchable documentation.
</Callout>

## Who Should Use PORMAZION

You benefit most if you manage technical docs, onboarding materials, or internal wikis. Developers document APIs, PMs outline roadmaps, and support teams build help centers—all in one platform.

<Expandable title="Brand Guidelines" default-open="true">
  Use PORMAZION's primary color `#3B82F6` for headers and accents. Pair with neutral grays for readability. Avoid altering the hue to maintain brand consistency across spaces.
</Expandable>

## Quick Start

Get up and running in minutes.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit `https://app.pormazion.com` and create your account with email or SSO.
  </Step>
  <Step title="Create a Space" icon="plus">
    Click "New Space" from the dashboard. Name it (e.g., "Project Alpha") and set permissions.
  </Step>
  <Step title="Add Your First Page" icon="file-plus">
    Inside the space, create a page. Use the editor for markdown, or import from Markdown/GitHub.
  </Step>
  <Step title="Invite Team" icon="mail">
    Share the space link or add members via email. Start collaborating immediately.
  </Step>
</Steps>
